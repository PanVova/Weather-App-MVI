# Weather-App-MVI
# The same as Weather-App-Kotlin but with (MVI + Epoxy)

Android Kotlin application to read api from https://www.metaweather.com/ using Retrofit


Used Technologies:
- Retrofit (OkHTTP + GSON)
- Coroutines
- Flow
- Dagger 2
- MVVM + MVI
- Clean Architecture
- Timber
- Epoxy
